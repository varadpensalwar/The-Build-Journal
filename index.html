Porsche 911 GT3 RS

Built a landing page inspired by the Porsche 911 GT3 RS with a 3D video carousel that rotates automatically, smooth animations, and a responsive design that looks good on desktop, tablet, and mobile screens. Used HTML, CSS, and JavaScript to create the carousel animation, responsive layout, and overall user experience. Deployed the project on Vercel.

AI Startup Outreach Tracker

Managing outreach to dozens of AI startups can quickly become difficult, so I built a single place to keep everything organized. The application lets users track each company's outreach status, set priorities, search records, and monitor overall progress without juggling spreadsheets. To make the experience practical, I added automatic data saving with Local Storage so updates persist between sessions, along with CSV export for sharing or maintaining offline records. The project was built using React, TypeScript, Tailwind CSS, and Vite.

Balaji Dresses & Saree Centre

The project began with a simple idea: a fashion website should feel as refined as the brand it represents. Instead of relying on static sections, I focused on creating a smooth, visually engaging experience where animations and transitions naturally guide users through the content. Built the experience using React, Vite, Styled Components, GSAP, Framer Motion, and Locomotive Scroll, combining responsive layouts with scroll-driven animations to deliver a polished interface across devices.
